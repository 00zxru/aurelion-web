# AURELION

A selective creative entity that observes and recognizes creatives based on precision, discipline, and execution.

## Identity

AURELION is not a platform or community. It is a standard that observes and recognizes creatives based on precision, discipline, and execution.

## Core Principles

- **Minimal, refined, controlled**
- **Mysterious and selective**
- **No marketing tone**
- **No explanations**
- **Everything feels intentional**

## Visual System

- **Colors**: Deep black, matte antique gold, soft white
- **Materials**: Matte black textile/suede/technical fabric
- **Lighting**: Soft directional lighting, no glossy textures
- **Atmosphere**: Minimal star field, slow drifting particles, subtle solar glow

## House System

AURELION observes through five distinct Houses:

- **HOUSE SOLIS** (Core / Vision)
- **HOUSE HELIOS** (Photography)
- **HOUSE APOLLO** (Music)
- **HOUSE VULCAN** (Fashion / Design)
- **HOUSE NOCTIS** (Experimental)

## Access

Recognition is required. Use access code: `AURELION2024`

## Development

Built with Next.js, TypeScript, TailwindCSS, and Framer Motion.

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Structure

- `/src/app` - Next.js app directory
- `/src/components` - React components
- `/src/types` - TypeScript type definitions
- `/access` - Creator access portal
- `/dashboard` - Creator dashboard

## Experience
